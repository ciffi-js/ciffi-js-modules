# ciffi-js-modules

JS Modules

- install with npm

```
npm install --save @ciffi-js/moduleName;
```

- install with yarn

```
yarn add @ciffi-js/moduleName;
```

- use

```javascript
import Module from "@ciffi-js/moduleName";
```

read more from each module

- [Cookies](packages/Cookies/README.md)
